# APOE4 API

## Installation

```bash
    npm install
```
```bash
    cp .env.sample .env
    npm run watch
```

## Documentation
Visit [APIDOCJS](http://apidocjs.com/) to learn how to use doc blocks. You can also check the AuthController for an existing documentation format.

```bash
    npm run apidoc
```
